# 第7章　广播
![](https://github.com/HBU/AndroidDemo/blob/master/chapter07/3.png)
## 作用
- 用于监听 / 接收 应用发出的广播消息，并做出响应
## 应用场景
- a. 不同组件之间通信（包括应用内 / 不同应用之间）
- b. 与 Android 系统在特定情况下的通信 如当电话呼入时、网络可用时
- c. 多线程通信
